**BL-02 Implementation Summary**

**Files changed**

[**dyfo/config.py**](vscode-webview://1dprqpp6k9pj4esjv7hdlfbbjkr6qng5rc79eskc0goecebaupf6/dyfo/config.py) - Added to DyFOConfig:

- model_variant: str = "tgn" - selects which encoder to use
- gat_num_layers: int = 2 - shared GAT depth
- roland_snapshot_freq: str = "monthly" - snapshot cadence
- roland_ema_alpha: float = 0.5 - EMA blend weight

**New files**

[**dyfo/core/model_variants.py**](vscode-webview://1dprqpp6k9pj4esjv7hdlfbbjkr6qng5rc79eskc0goecebaupf6/dyfo/core/model_variants.py) - Common interface:

- BaseGraphEncoder(nn.Module, ABC) - abstract base with reset_state(), advance_day(), get_node_embeddings(), detach_state() (default no-op)
- TGNWrapper - adapts existing DyFOModule to the interface; detach_state() implements TBPTT by detaching TGN memory after backward
- build_encoder(config, num_nodes, variant=...) - factory function

[**dyfo/core/gat_static_baseline.py**](vscode-webview://1dprqpp6k9pj4esjv7hdlfbbjkr6qng5rc79eskc0goecebaupf6/dyfo/core/gat_static_baseline.py) - GATStaticEncoder:

- 2-layer multi-head GAT over a fixed adjacency
- set_static_graph_from_correlations(corr_labels_by_date, train_dates) - builds the static graph from mean |ρ| over the training window (no look-ahead leakage)
- advance_day / reset_state are no-ops; detach_state inherits the base no-op

[**dyfo/core/roland_baseline.py**](vscode-webview://1dprqpp6k9pj4esjv7hdlfbbjkr6qng5rc79eskc0goecebaupf6/dyfo/core/roland_baseline.py) - ROLANDLikeEncoder (clearly documented as ROLAND-_like_, not a faithful re-implementation):

- precompute_monthly_snapshots(corr_labels_by_date) - builds per-month sparse edge indices
- advance_day detects month transitions and switches the active snapshot
- get_node_embeddings runs GAT(G_month, x_today) then applies EMA blend: h_new = (1-α)·h_prev.detach() + α·h_gat
- State \_h_prev is always stored detached (TBPTT analogue)

**Files refactored**

[**scripts/train_link_prediction.py**](vscode-webview://1dprqpp6k9pj4esjv7hdlfbbjkr6qng5rc79eskc0goecebaupf6/scripts/train_link_prediction.py):

- New model_variant parameter ("tgn" default, backward-compatible)
- DyFOModule replaced with build_encoder() factory throughout

- module.reset_memory() → encoder.reset_state()
- process_day_events() + compute_embeddings() → advance_day() + get_node_embeddings()
- module.encoder.memory.detach() → encoder.detach_state() (called after loss.backward())
- Checkpoint keys updated: module_state → encoder_state, model_variant added
- \_\_main\_\_now accepts --variant tgn|gat_static|roland CLI arg
